jdbc(Java Database Connectivity)

  • 含义

    即就是用java链接数据库执行sql语句的技术

  • 设计理念

    定义了一些接口（在java.sql包中），各个数据库厂商实现接口（Driver,Connnetion,State,resultSet）

  • 使用步骤

    • 在对应的官网下载驱动jar包(jar包中的类实现了上文中所说的关于jdbc的一些接口)

      比如在百度中直接搜索（mysql jdbc driver download）

    • 项目与模块引用（导入）jar包

    • 加载驱动程序Class.forName()

    • 建立连接Connection DriverManager.getConnection(url)

    • 创建Statement对象 con.createStatement()

    • 执行sql

      1. sta.executeUpdate() insert,update,dete (此方法为int 类型返回sql语句影响的行数)

      2. ResultSet executeQuery(sql) 此方法执行select语句，返回的是一个结果集

    • 关闭资源

  • DAO（Data Access Object）数据访问对象（持久化层设计思想）

    1. 包 com.yourname.dao 其中是具体的模块类DAO 四个方法（save update delete query）
    2. com.yourname.test 具体的模块类DAOtest
